Org Overview

A single view across every AI agent the platform runs. Each agent generates outputs (emails, offers, support replies), evaluates them automatically, and then reflects — proposing targeted improvements to its own prompt, skill files, and evaluation cases. This page tracks how those agents are performing right now, how they've improved over time, and what the platform has learned across all of them at once.

How scores are computed: Each agent runs a Langfuse experiment against its eval dataset. An LLM judge scores every output on domain dimensions (e.g. writing quality, personalisation, groundedness for B2B Sales); 4 deterministic heuristics run in parallel. The mean across all scorers is the row's overall score. Dimensions differ per agent — Customer Care uses empathy and resolution clarity instead of personalisation.

Intelligence Summary · May 2025

Across 7 campaign runs and 18 reflections, the portfolio improved by an average of +0.074 score points per reflection cycle. B2B Sales leads the portfolio at 0.94 after a single reflection that fixed opener framing and tone. A systemic "generic opener" pattern is active across all 3 agents — a shared prompt fix could accelerate the entire portfolio without individual reflection cycles. Customer Care shows the fastest recovery velocity after a rollback event (+0.12 in refl_cc_2). No previously-fixed failure has regressed across any agent in the last 3 runs.
